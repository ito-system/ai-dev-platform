# Skill: generate-saas-idea

## Responsibility
Generate a SaaS product idea given a target market and problem domain.

## Input
- `market`: Target market or industry (e.g., "HR teams at SMBs")
- `problem_domain`: Area of pain to solve (e.g., "employee onboarding")

## Steps
1. Identify the core pain point in the problem domain for the given market.
2. Generate a SaaS product concept that solves it.
3. Define the value proposition in one sentence.
4. List 3 core features that deliver the value proposition.
5. Identify the primary monetization model.

## Output
- Product name (placeholder)
- Value proposition
- 3 core features
- Monetization model

## Notes
- Ideas must be feasible for a small team to build.
- Avoid ideas requiring hardware or regulatory approval.
