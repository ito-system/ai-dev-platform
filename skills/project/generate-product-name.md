# Skill: generate-product-name

## Responsibility
Generate a product name based on the product concept and target audience.

## Input
- `concept`: One-sentence description of what the product does
- `audience`: Target user persona or market segment
- `style` (optional): Naming style — `descriptive`, `abstract`, `playful` (default: `descriptive`)

## Steps
1. Read the concept and audience.
2. Generate 5 candidate product names.
3. For each candidate, provide a one-sentence rationale.
4. Recommend the strongest option with justification.

## Output
- 5 candidate names with rationale
- 1 recommended name with justification

## Notes
- Names must be memorable, easy to spell, and domain-friendly.
- Avoid generic names like "AppSuite" or "ProTools".
